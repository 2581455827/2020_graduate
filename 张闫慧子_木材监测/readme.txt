#接线
	温湿度：
		SDA接P07
		SCL接地
	气体：
		AO接P06
	火光:
		DO接P11
	雨滴:
		DO接P13
	 #协调器
	
		RFID:NSS->P20
			SCK->P07
			MOSI->P06
			MISO->P05
			IRQ->不接
			GND->GND
			RST->P04
			VCC->3.3
#串口数据格式
	1122334566666666
	11温度	22湿度 33气体 4有无火灾 5有无雨滴	66666666卡号，如果没检测到卡就是00开头
#上位机要求
	刷RFID卡的目的是，每一个RFID卡号存着相关的木材信息，比如一批木材到木材厂后，刷一下卡，可以查看该批木材的信息，如重量，品种，运输时间，运输地点，司机等等信息
然后可以在上位机设置本次运送木材任务完成。
