#数据格式
	112233445555
	11 flash内设定的温度阈值最低值
	22 flash内设定的温度阈值最高值
	33实际温度值
	44实际湿度值
	5555光强
#使用说明
	本系统的温湿度范围需要上位机从下位机读取，具体见数据格式
	光强低于500，自动开灯
	本系统中含土壤湿度，但采用的是高低电平，所以没有数值，当过于干燥，自动开启水泵。
	本系统需使用蓝牙调试助手，连接蓝牙JDY-31-SPP ，密码1234，连接后串口显示“蓝牙连接成功”.断开则显示"蓝牙断开连接"，
当手机蓝牙向开发板发送数据#1234#,代表最低合适温度12度，最高合适温度34度，向flash内写入1234（必须带两个#号，且2个#之间的数据长度必须为4位）
