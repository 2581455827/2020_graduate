#接线
	温湿度：
		+接5v
		-接GND
		SDA接P0_7
		SCL接GND
	气体：AO接P0_6
		VCC-5v
		GND-GND
	led: +接P1_2
	       -接GND
	风扇:GND-GND
		VCC-3.3v
		INL-P1_1
		其他不接
	光照:GND-GND
		VCC-5V
		SCL-P0_5
		SDA-P0_4
		ADO-GND
#串口数据格式
	112233333444
	11温度，22湿度，33333光强，444气体浓度
#使用说明
	组网后，光照低于100，led自动亮，气体大于75，风扇转
	协调器按S2手动控制风扇，S1控制LED
	串口发送#1手动控制LED，#2控制风扇
