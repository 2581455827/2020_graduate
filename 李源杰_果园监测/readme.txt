#接线
	温湿度：
		SDA接P07
		SCL接地
	土壤:
		AO接P06
	LED  
		接P11
	水泵-继电器
		IN接P13
		COM接3.3V
;
		NC 接水泵红线
		水泵黑线接GND
	光照
		SCL P0_5
		SDA P0_4
		ADO  -》gnd
#串口数据格式
	11223334444
	11温度	22湿度 333土壤湿度  4444光照 
#使用说明
	光强低于400开灯
	土壤湿度大于110开启水泵(数值越大代表越干燥)
	串口发送AT1，开/关灯，进入手/自动模式，串口发送AT2，开/关水泵。进入手/自动模式
 #注意
	如果继电器频繁跳闸，可能是接线错误！！！
