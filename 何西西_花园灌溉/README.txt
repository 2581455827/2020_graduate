#数据格式
	1122333.3455556677
	11湿度
	22温度
	333.3土壤湿度，表示333.3%
	4自动/手动模式
	5555 光强
	66 flash内设定的最小土壤湿度
	777 flash内设定的最大土壤湿度
#使用说明
	注意，上位机需要通过串口来读取当前所设定的土壤湿度范围值，具体见数据格式
	光强低于400，自动开灯
	下位机有自动模式和手动模式，系统启动时，默认为自动监测模式
	自动检测模式下，土壤湿度大于flash设定的最大土壤湿度，也就是过于干燥，红灯亮，绿灯灭，开启水泵，蜂鸣器响；土壤湿度低于最大值时，红灯灭，绿灯亮，关闭水泵，关蜂鸣器
	按下KEY2，进行自动/手动模式切换，手动模式下，绿灯灭，红灯灭，开启水泵，关蜂鸣器
	手机可以通过蓝牙设置土壤湿度范围，格式为:AT1234,AT区分大小写，且最后要以\n（换行）结尾，蓝牙名：JDY-31-SPP 密码：1234,蓝牙连入时，会显示“蓝牙连接成功”，断开显示"蓝牙连接断开"
